# SwapMeet React — Phase 3 UI System Foundation (FREEZE)

## Objective
Define a complete UI system before redesigning any pages.

---

## Visual Direction
Warm modern marketplace (Etsy-adjacent, but cleaner and restrained)

- Soft off-white background
- White / light tan surfaces
- Charcoal text
- Muted accent colors
- Rounded cards
- Subtle shadows
- Image-forward layout

---

## Typography

**Fonts**
- Headings: Poppins or Montserrat
- Body: Inter

**Scale**
- Page Title: 32px / 700
- Section Title: 24px / 600
- Card Title: 18px / 600
- Body: 16px / 400
- Meta: 14px / 400
- Helper: 12px / 400

Rules:
- Max 3 weights per screen
- Prices stronger than descriptions

---

## Spacing (8px System)

4 / 8 / 12 / 16 / 24 / 32 / 48 / 64

Rules:
- Card padding: 16–24px
- Section gap: 32px
- Form gap: 16px

---

---

## Card System

- Radius: 14–18px
- Border: 1px soft
- Shadow: subtle
- Padding: 16–24px

Types:
- Product Card
- Category Card
- Dashboard Card

---

## Buttons

- Primary (filled)
- Secondary (outlined)
- Ghost (text)
- Danger (red)

Rules:
- One primary CTA per section
- Consistent radius

---

## Inputs

- White background
- Soft border
- Rounded corners
- Focus ring using primary color

Rules:
- Labels always visible
- Errors below input

---

## Layout

Container:
- Max width: 1200px
- Centered

Page Header:
- Title
- Optional action button

Grid:
- Responsive 1–4 columns

---

## Bootstrap Policy

Use for:
- Grid
- Layout utilities

Do NOT use for:
- Buttons
- Cards
- Forms
- Visual styling
